# LuxeMarket - E-commerce Website

A modern, nature-inspired e-commerce website built with Next.js 16.

## Features
- **Design System**: Sage green & terracotta palette with premium typography.
- **Pages**: Home, Product Detail, Cart, Checkout, About, Contact.
- **Mock Data**: Setup for Vercel deployment without database dependencies.

## Deployment
This project is configured to deploy on Vercel.
- Database dependencies (sqlite) have been removed.
- Uses in-memory mock data for demonstration.

## Getting Started
1. `npm install`
2. `npm run dev`
